# ChatAutoMod

A Bukkit plugin to automate punishment for restricted words

## Includes:
- Chat moderation (duh)
- Moderation for sign text

## How to use:
1. Build the project with gradle
2. Copy the "-all" jar into your plugins directory
3. Create a `ChatAutoMod` directory
4. Copy the `config` directory (not the contents, the whole directory) into `ChatAutoMod`

## Features:
- Fully customizeable severity levels
- Automatically executing punishments

Read the wiki for configuration info
